**Deskripsi Aplikasi BersihIn**

BersihIn merupakan aplikasi berbasis web yang dirancang untuk membantu masyarakat dalam mengelola proses penjemputan sampah rumah tangga secara lebih mudah, cepat, dan terorganisir. Melalui aplikasi ini, pengguna dapat melakukan registrasi akun, login, mengajukan permintaan penjemputan sampah, serta memantau status pengajuan secara langsung. Selain itu, setiap sampah yang berhasil dijemput akan dikonversikan menjadi poin reward berdasarkan berat sampah yang disetorkan. Poin tersebut dapat menjadi bentuk apresiasi kepada pengguna agar lebih peduli terhadap kebersihan lingkungan. Di sisi lain, admin memiliki hak akses untuk mengelola data pengguna, memverifikasi setiap permintaan penjemputan, memperbarui status pengajuan, serta memberikan poin reward kepada pengguna. Aplikasi ini dikembangkan menggunakan bahasa pemrograman PHP, database MySQL, serta framework Bootstrap untuk menghasilkan tampilan yang responsif dan mudah digunakan. Dengan adanya BersihIn, diharapkan proses pengelolaan sampah menjadi lebih efektif, efisien, dan mampu meningkatkan partisipasi masyarakat dalam menjaga kebersihan lingkungan.
